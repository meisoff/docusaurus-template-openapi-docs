# Первоначальные проверки

Перед созданием документа система партнера может провести первоначальные проверки:

### Проверить наличие пользователя в nopaper

Для данной проверки необходимо выполнить запрос `«Проверить наличие пользователя в nopaper»` — `partner-api/api/v2/external/profile-fl/user-guid/by-phone` . В запросе требуется передать номер телефона пользователя. 

В ответе nopaper вернёт `идентификатор пользователя`, если он зарегистрирован.

**Примечания:**
- Минимальная длина номера телефона - 8 цифр (сервис поддерживает регистрацию с ин. номерами)

### Проверить наличие компании в nopaper

Для данной проверки необходимо выполнить запрос `«Проверить наличие компании»` — `partner-api/api/v2/external/company/by-inn` . В запросе требуется передать ИНН компании. 

В ответе nopaper вернёт список компаний с краткой информацией (идентификатор компании, КПП, короткое название).

**Примечания:**
- Реквизиты для юридического лица — ИНН и КПП; для ИП — только ИНН.

### Получить список компаний где пользователь является сотрудником

Чтобы получить список компаний с указанным сотрудником необходимо выполнить запрос `«Получить список компаний где пользователь является сотрудником»` — `partner-api/api/v2/external/profile-fl/company/list`. В запросе требуется передать `номер телефона пользователя` или его `email` (на выбор). 

В ответе nopaper вернёт список компаний с краткой информацией (идентификатор компании, КПП, короткое название).
